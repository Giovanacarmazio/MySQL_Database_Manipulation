# MySQL Database Manipulation

Este repositório contém o projeto **MySQL Database Manipulation**, focado em demonstrar como interagir com bancos de dados MySQL utilizando Python. O projeto cobre as operações básicas de criação, leitura, atualização e exclusão (CRUD) de registros no banco de dados, facilitando o gerenciamento de dados através de scripts automatizados.

## Descrição do Projeto

O objetivo do **MySQL Database Manipulation** é ensinar como conectar-se a um banco de dados MySQL, executar consultas SQL e manipular dados de maneira eficiente utilizando Python.

## Funcionalidades

- Conexão com um banco de dados MySQL.
- Criação de tabelas e inserção de registros.
- Execução de consultas SQL para leitura, atualização e exclusão de dados.

## Tecnologias Utilizadas

- **Python**
- **MySQL**: Banco de dados relacional.
- **MySQL Connector**: Biblioteca Python para conectar ao MySQL.

## Estrutura do Repositório

- `db_manipulation.py`: Código principal que executa as operações no banco de dados.
- `docs/`: Documentação sobre as instruções de instalação e uso.

## Como Executar

### Pré-requisitos

- MySQL instalado e configurado.
- Python 3.7+.
- Instalar dependências:
   ```bash
   pip install mysql-connector-python
   ```

### Passos para execução

1. Clone o repositório:
   ```bash
   git clone https://github.com/Giovanacarmazio/Manipulacao-MYSQL-BD.git
   ```

2. Execute o script:
   ```bash
   python db_manipulation.py
   ```

3. Insira os dados de conexão e execute as operações SQL desejadas.
